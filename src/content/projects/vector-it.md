VectorIt is an open-source desktop app that converts raster images into vector graphics. It accepts PNG, JPG, BMP, GIF, and TIFF input and produces SVG, EPS, PDF, and DXF vector output. The app is built with Rust, uses Tauri v2 for a "featherweight" desktop shell of roughly 5 MB, and is wrapped in a React + TypeScript frontend. It runs on Windows, macOS, and Linux, with builds for each platform published on the GitHub Releases page.
