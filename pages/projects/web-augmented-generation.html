Web Augmented Generation is a Node.js application that enhances LLM responses by incorporating real-time web search results. It uses SearXNG for web searches and supports various LLM providers through OpenAI-compatible API calls, enabling more accurate and up-to-date responses.

GitHub

Key Features

  • Query optimization for web searching
  • Web search integration via SearXNG
  • Content fetching and summarization
  • Support for multiple LLM providers
  • Streaming response support
  • Content similarity checking
  • Interactive CLI interface
  • Comprehensive error handling
