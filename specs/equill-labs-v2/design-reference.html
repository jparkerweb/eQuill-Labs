semantic-chunking

A semantic-aware text splitter for retrieval pipelines.

Classic character- and token-based chunking shreds ideas mid-sentence. The semantic-chunking library embeds candidate break points and clusters adjacent sentences, so a chunk stays about one thing — yielding cleaner retrieval, better grounding, and fewer hallucinations downstream.

Works with any embedding provider; ships with sane defaults for OpenAI, Cohere, and local via Transformers.js. Used in production by several RAG systems.

\\ Install
$ npm install semantic-chunking
\\ Quick use
import {'{ chunker }'} from
  'semantic-chunking';
await chunker(text, {'{ size: 512 }'});
